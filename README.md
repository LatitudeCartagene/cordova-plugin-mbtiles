cordova-plugin-mbtiles 3.4.0
======================

To install plugin 
----------------------

* Android

create project by cordova 

	cordova create <project_name> <package_name> <name_class>

Then

	cordova platform add android

	cordova add plugin <path_to_src>

* iOS

create project by cordova 

	cordova create <project_name> <package_name> <name_class>

Then

	cordova platform add ios

	cordova add plugin <path_to_src>
